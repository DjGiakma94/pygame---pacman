Requisiti Python 3.6+ (per il corso sto usando 3.10 ma non sfrutterò le funzioni più recenti)

installare l'ultima versione di pygame con:
pip install pygame

installare Visual Studio Code e la sua extension chiamata "Python" fatta da Microsoft (ce ne sono altre, ma preferisco quella ufficiale)

Aprire il progetto con File -> Open Folder (andate nella cartella dove avete fatto il clone del repo)

Selezionate un file da eseguire e premete F5, selezionate Python Debugging, per avviare una sessione di Debug.

Enjoy!
